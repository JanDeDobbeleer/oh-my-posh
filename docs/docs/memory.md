## Memory

Display memory usage percentage.

## Sample Configuration

```json
{
  "type": "memory",
  "style": "powerline",
  "foreground": "#26C6DA",
  "background": "#2f2f2f",
  "properties": {
    "precision": 1,
    "prefix": " \uf85a ",
    "postfix": "% "
  }
}
```

## Properties

- precision: `int` - the number of decimal places to show - defaults to `0`
