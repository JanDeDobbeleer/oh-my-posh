## What

Show when the current user is root or when in an elevated shell (Windows).

## Sample Configuration

```json
{
  "type": "root",
  "style": "powerline",
  "powerline_symbol": "\uE0B0",
  "foreground": "#111111",
  "background": "#ffff66",
  "properties": {
    "template": "\uF0E7"
  }
}
```
