# Oh My Posh — Fish Setup

> **Requires Fish 4.1.0 or later.** Older versions have issues and do not support
> transient prompt.

## Configure

Add this as the **last line** of `~/.config/fish/config.fish`:

```bash
oh-my-posh init fish | source
```

Reload:

```bash
exec fish
```
