# Oh My Posh — Nu / Nushell Setup

> **Requires Nushell v0.104.0 or later.**

## Configure

Add this as the **last line** of the Nushell config file (path: `$nu.config-path`):

```bash
oh-my-posh init nu
```

Restart Nushell to apply.
